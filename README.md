# CodeV_Macros
这是一个用于CodeV软件的宏脚本集合，旨在简化和自动化常见任务，提高工作效率。

## 安装
1. 下载本仓库的代码。
2. 将所需的宏脚本文件复制到CodeV的宏目录中，通常位于`C:\Program Files\CodeV\Macros`。
3. 重启CodeV软件以加载新的宏脚本。
4. 在CodeV中，导航到“工具”菜单，选择“宏”，然后选择所需的宏脚本运行。
## 使用说明
1. 打开CodeV软件。
2. 导航到“工具”菜单，选择“宏”。
3. 选择所需的宏脚本并点击“运行”。
4. 根据宏脚本的提示进行操作。
5. 查看宏脚本的输出结果。
## 宏列表
- **自动对齐光学元件**：自动调整光学元件的位置以实现最佳对齐。
- **批量导出光学设计**：将多个光学设计文件批量导出为指定格式。
- **光学性能分析**：自动运行光学性能分析并生成报告。
- **镜头设计优化**：根据预设参数自动优化镜头设计。
- **光学系统仿真**：自动设置并运行光学系统的仿真。
- **数据可视化**：生成光学设计数据的可视化图表。
